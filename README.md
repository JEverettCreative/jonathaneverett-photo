# Jonathan Everett Creative — Nashville

Photography portfolio built with [Eleventy (11ty)](https://www.11ty.dev/).

## Setup

```bash
npm install
npm start        # Dev server at http://localhost:8080
npm run build    # Build to _site/
```

## Pages

- **/** — Home (hero, featured work, CTA)
- **/photography/physique/** — Physique Photography (competition, transformation, pricing)
- **/photography/headshots/** — Headshots (acting, modeling, business + pricing)
- **/photography/portraits/** — Portraits & Creative (cosplay, conceptual, editorial)
- **/film/** — Film & Acting work (video embeds)
- **/blog/** — Blog listing
- **/about/** — Bio
- **/contact/** — Contact form + info

## Stack

- Eleventy 2.x with Nunjucks templates
- Custom CSS (no framework)
- Vanilla JS lightbox + mobile nav
- Dark theme with gold accent (#d4a843)
